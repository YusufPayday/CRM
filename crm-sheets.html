Kevytyrittäjyys CRM

Syötä salasana jatkaaksesi.

Tervetuloa — aloitetaan

Yhdistä CRM Google Sheetiisi. Tarvitset Apps Script -URL:n ja valitsemasi salaisen tokenin, jonka liität myös Code.gs-tiedostoon.

Vaihe 1 · Sheet

Avaa Google Sheet

Avaa tai luo Sheet → Laajennukset → Apps Script → liitä Code.gs-tiedoston sisältö editoriin.

Vaihe 2 · Token

Luo salainen token

Tämä estää ulkopuolisia käyttämästä API:asi.

Vaihe 3 · Code.gs

Liitä token Code.gs-tiedostoon

Etsi Code.gs:n alusta rivi const TOKEN = … ja korvaa se tällä:

const TOKEN = '';

Tallenna (Cmd/Ctrl + S) → Käyttöönotto → Hallitse käyttöönottoja → kynäikoni → Versio: Uusi versio → Ota käyttöön.

Vaihe 4 · URL

Liitä Apps Script -URL

Käyttöönoton jälkeen saat URL:n joka päättyy /exec.
